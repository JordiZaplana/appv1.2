Sobre la app

Esta App forma parte de un sistema formado por una base de datos y dos microcontoladors que interactuan con la app.

Se ha dispuesto de una Raspberry Pi 3 para la captura de imagenes y detección de movimiento. También se ha usado un esp32 thing de sparkfun con un shield de sensores. Los sensores utilizados en este proyecto són:

  • Temperatura
  • Presión
  • Calidad del aire(cantidad de co2)
  • Humedad
  • Luminosidad
